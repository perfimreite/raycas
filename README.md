# Raycas 

Raycas is a game engine built using the raycasting rendering technique to create a 3D perspective in a 2D map.

## Sources

- https://lodev.org/cgtutor/raycasting.html

## Dependecies

Raycas depends on SDL2.

```console
$ sudo apt install libsdl2-2.0-0 libsdl2-dev libsdl2-ttf-2.0-0 libsdl2-ttf-dev libsdl2-image-2.0-0 libsdl2-image-dev
```
